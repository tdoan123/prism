# Prism

a platform that helps analysts uncover the meaning behind boring documents

---

## Why Prism?

Due diligence analysts spend 80% of their time moving data between 
Excel, Word, and PowerPoint — and 20% actually thinking. 
Prism flips that ratio.

---

## Build in Public

This project is built in public as part of BCIT's Software Systems 
Development program 2025 - 2026. All architectural decisions, 
tradeoffs, and learning moments are documented as the project progresses.
